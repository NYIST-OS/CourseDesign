# CourseDesign

## 关于

此仓库内的项目均是本人在校期间所做的小项目，仅为开源，共享，方便同学们交流。

> **如何下载？**
> <br>
> 点击 Download ZIP 即可下载整个仓库（不支持手机版），由于众所周知的原因，下载速度略慢。
> <br>
> 如果你连 Download ZIP 都找不到...
> <br>
> 那么按 Ctrl + F 搜索页面中的 Download ZIP 并点击下载即可
> <br>
> **中文乱码？**
> <br>
> 请将项目的编码设置为 UTF-8，同理建数据库时都要保证为 UTF-8
